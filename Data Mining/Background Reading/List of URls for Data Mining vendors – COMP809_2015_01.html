        List of URls for Data Mining vendors

        Content

        • Oracle's product

          Oracle recently acquired Thinking Machines Corp - one of the pioneers in the field of Mining. They  incoporate Darwin (Thinking Machines Data Mining suite) into their  Database engine from Oracle version 9 onwards.

        • Attar Software

          Specializes in financial data analysis.

        • Business Objects

          Their product is BusinessMiner which uses a Decision Tree approach.

        • NeuralWare Inc

          Specialises in Neural Network algorithms for Data Mining

        • SPSS Inc.

          Traditionally a vendor of Statistical Analysis software but has support for Data Mining through their Clemntine and AnswerTree products.

        • SAS Inc

          A vendor of analytical software who has incorporated data mining software and markets it as SAS Enterprise Miner.

        • Microsoft Corporation

          Microsoft has integrated Data Mining products into the their SQL Server 2008 platform.

        • Weka Machine Learning Suite

          Although not an actual vendor in the actual sense as it is free to download and use, it nevertheless needs to be maintained and this is done by the original developers at the Machine Learning Lab, University of Waikato, New Zealand. It is one of the most popular machine learning suites in terms of usage world-wide, both in terms of Industry usage and usage amonst machine learning researchers. This is due to the fact that it has one of the most compreshensive machine learning suites currently available.

        • The R Package

          The R package, like many other statistical analysis packages started off as a conventional data analsyis tool. It has now added functionality for data mining and is increasing in popularity word-wide. Just as with Weka, it is available free of charge for download.
